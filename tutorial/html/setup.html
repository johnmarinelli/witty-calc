Building a calculator with WT

What is Wt?

Wt is a widget-centric web framework that uses C++ and WebSockets to build web apps. More information can be found at their homepage.

Motivation

Wt has a plethora of example code explaining how to use a lot of its features; however, it has few tutorials that guide you through the framework. While this may suffice for languages that are already established for web development, C++ has a notorious reputation for being "hard". The purpose of this tutorial is to draw some parallels between a typical web app written in an interpreted language and Wt.

I assume you already know...

  • Some basic C++ (pointers, classes, inheritance)
  • How to compile and link C++ programs with g++ (a build script will be provided and explained, but you should know what 'g++ main.cpp -o hello' means)
  • The Wt 'Hello World' tutorial

This tutorial will show you:

  • How to encapsulate components of your app into multiple-use Widgets
  • How to aggregate different pieces of your app into a Widget
  • How to use Wt's powerful Signals & Slots system
  • How std::bind works, and how it works with the Signals & Slots system

Setup

You will need...

  • A compiler that supports C++11
  • The Wt framework installed on your machine (Wt relies on Boost, so you'll need that too)

This tutorial was written using GCC version 4.8.2 and Linux Mint 17 during Wt 3.3.3.

Wt should have a reasonably recent version in your package manager, as should Boost.
